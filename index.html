Misión de la carrera

Formar profesionales íntegros, capacitados en el desarrollo de software y soluciones tecnológicas, con enfoque en investigación aplicada y responsabilidad social.

Visión de la carrera

Ser una carrera reconocida por egresar profesionistas con competencias tecnológicas actuales, experiencia práctica y una actitud de aprendizaje permanente.

Objetivo de la materia

Desarrollar habilidades en HTML, CSS y JavaScript para diseñar y crear sitios web funcionales, accesibles y estéticamente atractivos.

Consideraciones sobre las sesiones

  • Clases de lunes a viernes según el horario.
  • Asistencia mínima del 90%.
  • Material disponible en Blackboard y/o Teams.
  • Entrega de tareas en la plataforma y dentro del plazo establecido.
  • No se aceptan tareas por correo.

Contenido de la materia

  1. Lenguaje HTML
  2. Lenguaje HTML avanzado
  3. Hojas de estilo CSS
  4. JavaScript
  5. Sitios responsivos
